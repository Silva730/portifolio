# Portfólio Pessoal — Primeiro Projeto

Este é o **primeiro projeto da minha carreira**, criado para colocar em prática meus conhecimentos iniciais em **frontend** e **manipulação com JavaScript**.

## Objetivo

Praticar os fundamentos de desenvolvimento web, estruturação de páginas, estilização básica e interações simples com JavaScript.

## Características

- Site estático e apenas visualizável
- Estrutura simples e direta
- Manipulação básica do DOM com JavaScript
- Layout feito com HTML e CSS básicos

## Tecnologias utilizadas

- HTML
- CSS
- JavaScript

## Observações

- Projeto inicial de aprendizado
- Código simples e sem complexidade
- Desenvolvido para prática e evolução técnica

## Status do projeto

- Tipo: Frontend
- Nível: Inicial
- Uso: Portfólio
